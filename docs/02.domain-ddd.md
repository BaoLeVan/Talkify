## 2.3 Messaging Domain — Chi tiết Aggregate Roots

### Aggregate Root: `Message`

```java
/**
 * Invariants:
 * - messageId phải unique (Snowflake ID)
 * - senderId phải là participant trong conversation
 * - content không được rỗng (trừ khi có attachment)
 * - sequenceNumber tăng đơn điệu trong mỗi conversation
 * - Không thể chỉnh sửa sau 24 giờ
 */
public class Message extends AggregateRoot<MessageId> {

    private final MessageId messageId;           // Snowflake-based
    private final ConversationId conversationId;
    private final UserId senderId;
    private final MessageContent content;
    private final MessageType type;              // TEXT, IMAGE, FILE, AUDIO, VIDEO, SYSTEM
    private final long sequenceNumber;
    private final Instant createdAt;
    private final MessageId replyToMessageId;   // nullable

    private MessageStatus status;               // SENT → DELIVERED → READ
    private boolean deleted;
    private Instant editedAt;
    private final List<DeliveryReceipt> deliveryReceipts = new ArrayList<>();

    public static Message create(ConversationId convId, UserId senderId,
                                  MessageContent content, MessageType type,
                                  long sequenceNumber) {
        var msg = new Message(MessageId.generate(), convId, senderId,
                              content, type, sequenceNumber, Instant.now());
        msg.registerEvent(new MessageSentEvent(
            msg.messageId, convId, senderId, content, type, Instant.now()
        ));
        return msg;
    }

    public void markDelivered(UserId recipientId, Instant deliveredAt) {
        deliveryReceipts.add(new DeliveryReceipt(recipientId, deliveredAt, null));
        registerEvent(new MessageDeliveredEvent(messageId, recipientId, deliveredAt));
    }

    public void markRead(UserId readerId, Instant readAt) {
        deliveryReceipts.stream()
            .filter(r -> r.recipientId().equals(readerId))
            .findFirst()
            .ifPresent(r -> r.markRead(readAt));
        registerEvent(new MessageReadEvent(messageId, conversationId, readerId, readAt));
    }

    public void softDelete(UserId deletedBy) {
        if (!senderId.equals(deletedBy))
            throw new DomainException("Chỉ người gửi mới có thể xóa tin nhắn");
        this.deleted = true;
    }

    public void edit(UserId editor, String newText) {
        if (!senderId.equals(editor))
            throw new DomainException("Chỉ người gửi mới có thể sửa tin nhắn");
        if (Duration.between(createdAt, Instant.now()).toHours() > 24)
            throw new DomainException("Không thể sửa tin nhắn sau 24 giờ");
        this.content = MessageContent.withText(newText, content.attachments(), content.metadata());
        this.editedAt = Instant.now();
    }
}
```

### Aggregate Root: `Conversation`

```java
/**
 * Invariants:
 * - DIRECT conversation có đúng 2 participants
 * - GROUP conversation có 2-N participants (N ≤ maxMembers)
 * - Không tồn tại 2 DIRECT conversation giữa cùng 2 users
 */
public class Conversation extends AggregateRoot<ConversationId> {

    private final ConversationId id;
    private final ConversationType type;
    private final Instant createdAt;

    private String title;
    private String avatarUrl;
    private final Set<Participant> participants = new HashSet<>();

    // Denormalized cho quick access
    private MessageId lastMessageId;
    private String lastMessagePreview;
    private Instant lastMessageAt;
    private long messageSequenceCounter;

    public static Conversation createDirect(UserId user1, UserId user2) {
        if (user1.equals(user2))
            throw new DomainException("Không thể tạo conversation với chính mình");
        var conv = new Conversation(ConversationId.generate(), ConversationType.DIRECT, Instant.now());
        conv.participants.add(Participant.member(user1));
        conv.participants.add(Participant.member(user2));
        conv.registerEvent(new ConversationCreatedEvent(conv.id, ConversationType.DIRECT));
        return conv;
    }

    public static Conversation createGroup(UserId creatorId, String title, Set<UserId> memberIds) {
        var conv = new Conversation(ConversationId.generate(), ConversationType.GROUP, Instant.now());
        conv.title = title;
        conv.participants.add(Participant.owner(creatorId));
        memberIds.forEach(id -> conv.participants.add(Participant.member(id)));
        return conv;
    }

    public long nextSequenceNumber() {
        // Thread-safe thông qua Redis INCR ở application layer
        return ++messageSequenceCounter;
    }

    public boolean isParticipant(UserId userId) {
        return participants.stream()
            .anyMatch(p -> p.userId().equals(userId) && p.isActive());
    }

    public void updateLastMessage(MessageId msgId, String preview, Instant sentAt) {
        this.lastMessageId = msgId;
        this.lastMessagePreview = preview;
        this.lastMessageAt = sentAt;
    }

    public void addMember(UserId adminId, UserId newMemberId) {
        Participant admin = requireAdmin(adminId);
        if (isParticipant(newMemberId))
            throw new DomainException("User đã là thành viên");
        participants.add(Participant.member(newMemberId));
        registerEvent(new MemberJoinedEvent(id, newMemberId, Instant.now()));
    }

    public void removeMember(UserId adminId, UserId targetId) {
        requireAdmin(adminId);
        participants.stream()
            .filter(p -> p.userId().equals(targetId))
            .findFirst()
            .ifPresent(Participant::deactivate);
        registerEvent(new MemberRemovedEvent(id, targetId, Instant.now()));
    }
}
```

### Value Objects

```java
// === MessageId — Snowflake ===
public record MessageId(long value) {
    public static MessageId generate() {
        return new MessageId(SnowflakeIdGenerator.nextId());
    }
}

// === MessageContent ===
public record MessageContent(String text, List<Attachment> attachments,
                              Map<String, String> metadata) {
    public MessageContent {
        if ((text == null || text.isBlank()) && (attachments == null || attachments.isEmpty()))
            throw new DomainException("Message phải có text hoặc attachment");
        if (text != null && text.length() > 10_000)
            throw new DomainException("Nội dung không vượt quá 10,000 ký tự");
    }

    public String preview(int maxLen) {
        if (text != null && !text.isBlank()) return text.substring(0, Math.min(text.length(), maxLen));
        if (attachments != null && !attachments.isEmpty()) return "[Tệp đính kèm]";
        return "";
    }
}

// === DeliveryReceipt ===
public record DeliveryReceipt(UserId recipientId, Instant deliveredAt, Instant readAt) {
    public DeliveryReceipt markRead(Instant readAt) {
        return new DeliveryReceipt(this.recipientId, this.deliveredAt, readAt);
    }
}

// === Attachment ===
public record Attachment(String fileId, String fileName, String mimeType,
                          long fileSize, String url, String thumbnailUrl) {}
```
