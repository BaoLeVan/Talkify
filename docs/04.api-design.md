# 04 — Thiết kế API

## 4.1 REST API

### Base URL: `/api/v1`
### Authentication: JWT Bearer Token (trừ auth endpoints)
### Response Format:
```json
{
  "success": true,
  "data": { ... },
  "error": null,
  "timestamp": "2026-03-03T10:15:30Z"
}
```

---

### 4.1.1 Authentication

| Method | Endpoint | Mô tả | Body |
|--------|----------|-------|------|
| POST | `/auth/register` | Đăng ký | `{username, password, displayName, phone?}` |
| POST | `/auth/login` | Đăng nhập → JWT | `{username, password}` |
| POST | `/auth/refresh` | Refresh access token | `{refreshToken}` |
| POST | `/auth/logout` | Đăng xuất | Header: Authorization |
| POST | `/auth/change-password` | Đổi mật khẩu | `{oldPassword, newPassword}` |

### 4.1.2 Users

| Method | Endpoint | Mô tả |
|--------|----------|-------|
| GET | `/users/me` | Profile của user đang login |
| PUT | `/users/me` | Cập nhật profile `{displayName, avatarUrl}` |
| PUT | `/users/me/avatar` | Upload avatar (multipart) |
| GET | `/users/{userId}` | Profile user khác |
| GET | `/users/search?q=&page=&size=` | Tìm kiếm user |
| POST | `/users/me/devices` | Đăng ký device token cho push |
| DELETE | `/users/me/devices/{deviceId}` | Xóa device token |
| GET | `/users/{userId}/presence` | Trạng thái online của user |

### 4.1.3 Conversations

| Method | Endpoint | Mô tả | Body/Params |
|--------|----------|-------|-------------|
| GET | `/conversations?page=&size=` | Danh sách conversations (sorted by lastMessageAt) | — |
| POST | `/conversations` | Tạo conversation | `{type, participantIds, title?}` |
| GET | `/conversations/{convId}` | Chi tiết conversation | — |
| PUT | `/conversations/{convId}` | Cập nhật title/avatar | `{title?, avatarUrl?}` |
| DELETE | `/conversations/{convId}` | Rời / Giải tán conversation | — |
| POST | `/conversations/{convId}/mute` | Tắt thông báo | `{duration: "1h"/"8h"/"forever"}` |
| DELETE | `/conversations/{convId}/mute` | Bật lại thông báo | — |
| GET | `/conversations/{convId}/participants` | Danh sách thành viên | — |
| POST | `/conversations/{convId}/participants` | Thêm thành viên (admin) | `{userIds: []}` |
| DELETE | `/conversations/{convId}/participants/{userId}` | Xóa thành viên (admin) | — |
| PUT | `/conversations/{convId}/participants/{userId}/role` | Đổi role | `{role: "ADMIN"/"MEMBER"}` |

### 4.1.4 Messages

| Method | Endpoint | Mô tả | Body/Params |
|--------|----------|-------|-------------|
| GET | `/conversations/{convId}/messages` | Lịch sử tin nhắn | `?before={seqNum}&limit=50` |
| GET | `/conversations/{convId}/messages/{msgId}` | Chi tiết tin nhắn | — |
| DELETE | `/conversations/{convId}/messages/{msgId}` | Xóa tin nhắn (soft) | — |
| PUT | `/conversations/{convId}/messages/{msgId}` | Sửa tin nhắn | `{text}` |
| POST | `/conversations/{convId}/messages/read` | Đánh dấu đã đọc | `{lastReadSequence}` |
| GET | `/conversations/{convId}/messages/search` | Tìm kiếm tin nhắn | `?q=&from=&after=&before=` |
| GET | `/conversations/{convId}/messages/{msgId}/receipts` | Danh sách người đã đọc | — |

### 4.1.5 Groups

| Method | Endpoint | Mô tả |
|--------|----------|-------|
| PUT | `/groups/{convId}/settings` | Cập nhật settings nhóm |
| POST | `/groups/{convId}/invite-link` | Tạo invite link |
| DELETE | `/groups/{convId}/invite-link` | Vô hiệu hóa invite link |
| POST | `/groups/join/{inviteCode}` | Tham gia nhóm qua link |
| GET | `/groups/{convId}/invitations` | Danh sách lời mời pending |
| POST | `/groups/{convId}/invitations` | Mời user vào nhóm `{userId}` |
| PUT | `/groups/{convId}/invitations/{invId}` | Chấp nhận/Từ chối `{action}` |
| POST | `/groups/{convId}/transfer-ownership` | Chuyển quyền owner |

### 4.1.6 Media

| Method | Endpoint | Mô tả |
|--------|----------|-------|
| POST | `/media/upload` | Upload file (≤ 50MB, multipart) |
| POST | `/media/upload/presigned` | Lấy presigned URL (file lớn, S3 direct) |
| GET | `/media/{fileId}` | Download / stream file |
| DELETE | `/media/{fileId}` | Xóa file |

### 4.1.7 Offline Sync

| Method | Endpoint | Mô tả |
|--------|----------|-------|
| GET | `/sync?since={timestamp}&limit=200` | Delta sync khi user online lại |
| POST | `/sync/ack` | Xác nhận đã nhận đủ `{lastSyncedMessageId}` |

## 4.2 WebSocket Events (STOMP)

### Connection
```
# Client kết nối
CONNECT
Authorization: Bearer {jwt_token}
heart-beat: 30000,30000

# Server phản hồi
CONNECTED
version: 1.2
heart-beat: 30000,30000
user-name: {userId}
```

---

### Client → Server Events

#### 1. Gửi tin nhắn
```
SEND
destination: /app/chat.send
content-type: application/json

{
  "clientMessageId": "c4f3d2a1-...",      // UUID — idempotency key
  "conversationId": "conv-abc-123",
  "type": "TEXT",                          // TEXT | IMAGE | FILE | AUDIO | VIDEO
  "content": {
    "text": "Xin chào mọi người!",
    "attachments": [
      {
        "fileId": "f001",
        "fileName": "photo.jpg",
        "mimeType": "image/jpeg",
        "fileSize": 204800,
        "url": "https://cdn.chat.com/f001",
        "thumbnailUrl": "https://cdn.chat.com/f001/thumb"
      }
    ],
    "metadata": {}
  },
  "replyToMessageId": null
}
```

#### 2. Đang gõ
```
SEND
destination: /app/chat.typing

{ "conversationId": "conv-abc-123", "isTyping": true }
```

#### 3. Đánh dấu đã đọc
```
SEND
destination: /app/chat.read

{ "conversationId": "conv-abc-123", "lastReadSequence": 1501 }
```

#### 4. Delivery ACK (client xác nhận đã nhận)
```
SEND
destination: /app/chat.ack

{ "messageIds": [7281946281, 7281946282] }
```

#### 5. Cập nhật Presence
```
SEND
destination: /app/presence.update

{ "status": "AWAY" }    // ONLINE | AWAY | DO_NOT_DISTURB
```

#### 6. Subscribe conversations
```
SUBSCRIBE
destination: /user/queue/messages
id: sub-personal

SUBSCRIBE
destination: /user/queue/errors
id: sub-errors
```

### Server → Client Events

#### 1. MESSAGE_ACK — Server xác nhận nhận được
```json
{
  "event": "MESSAGE_ACK",
  "clientMessageId": "c4f3d2a1-...",
  "serverMessageId": 7281946281,
  "sequenceNumber": 1501,
  "status": "SENT",
  "timestamp": "2026-03-03T10:15:30Z"
}
```

#### 2. NEW_MESSAGE — Có tin nhắn mới
```json
{
  "event": "NEW_MESSAGE",
  "messageId": 7281946281,
  "conversationId": "conv-abc-123",
  "senderId": 1001,
  "senderName": "Nguyễn Văn A",
  "senderAvatar": "https://cdn.chat.com/avatar/1001",
  "sequenceNumber": 1501,

  "content": {
    "text": "Xin chào mọi người!",
    "attachments": [],
    "metadata": {}
  },

}
```
